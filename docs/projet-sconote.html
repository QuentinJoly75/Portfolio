Sconote

Application web de gestion de notes pour les étudiants

Description du projet

Sconote est un projet de groupe qui a été réalisé par 4 collaborateurs et moi. Ce site a pour bût de permettre aux professeurs de l’IUT, de gérer les notes des élèves, pour préparer les conseils du 5ème semestre. Les élèves peuvent ainsi donner accès à des écoles externes de voir leurs notes, dans le but de les recruter en sorti de BUT. Ils peuvent aussi lire des avis donnés par les professeurs, qui sont visible seulement après le conseil.

L'application permet aux utilisateurs de :

  • Enregistrer et organiser les notes de cours par matière
  • Suivre ses résultats scolaires et calculer ses moyennes
  • Gérer un calendrier d'échéances pour les devoirs et examens
  • Partager des ressources avec d'autres étudiants
  • Recevoir des notifications pour les échéances importantes

Ce projet a été développé dans le cadre de mon parcours en BUT Informatique, avec une approche centrée sur l'expérience utilisateur et l'accessibilité.

Technologies utilisées

HTML5 CSS3 JavaScript PHP MySQL Bootstrap Git

Pour ce projet, j'ai choisi d'utiliser PHP comme langage back-end pour sa facilité d'intégration avec MySQL et sa large communauté. La base de données MySQL stocke les informations des utilisateurs, les notes, et les échéances de manière structurée et sécurisée.

Le front-end est construit avec HTML5, CSS3 et JavaScript, avec l'aide du framework Bootstrap pour assurer une interface responsive et cohérente sur tous les appareils. L'architecture du projet suit le modèle MVC (Modèle-Vue-Contrôleur) pour une meilleure organisation et maintenabilité du code.

Tout le code est versionné avec Git, permettant un développement itératif et collaboratif efficace.
