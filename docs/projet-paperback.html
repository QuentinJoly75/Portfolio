Paperback's Extensions (WIP)

Extensions pour l'application de lecture Paperback (en cours de développement)

Description du projet

Paperback's Extensions est un projet de développement d'extensions pour l'application de lecture Paperback, un lecteur de mangas open source pour iOS. Ces extensions permettent d'accéder à diverses sources de contenu et d'enrichir l'expérience utilisateur de l'application principale.

Mon travail sur ce projet a consisté à :

  • Développer des API wrapper pour différentes sources de contenu
  • Optimiser les performances des requêtes et du parsing des données
  • Implémenter des fonctionnalités de recherche et de filtrage avancées
  • Assurer la compatibilité avec les mises à jour de l'application principale
  • Collaborer avec la communauté open source pour améliorer continuellement les extensions

Ce projet m'a permis de travailler sur un produit réel avec une communauté active d'utilisateurs, et d'approfondir mes connaissances en développement d'API et en manipulation de données web.

Technologies utilisées

TypeScript Node.js API REST JSON Git GitHub Actions

Pour ce projet, j'ai principalement travaillé avec TypeScript, qui offre une expérience de développement robuste grâce à son système de typage statique. Cela s'est avéré particulièrement utile pour maintenir la cohérence des données à travers les différentes parties du code.

Le développement s'est fait en interfaçant avec diverses API REST pour récupérer et traiter les données des sources. J'ai implémenté des systèmes de cache intelligents pour optimiser les performances et réduire la charge sur les serveurs externes.

J'ai également mis en place des workflows avec GitHub Actions pour automatiser les tests et le déploiement des extensions, garantissant ainsi leur qualité et leur disponibilité rapide pour les utilisateurs.
