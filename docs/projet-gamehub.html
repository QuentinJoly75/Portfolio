GameHub

Plateforme web de mini-jeux rétro et classiques

Description du projet

GameHub est une plateforme web que j'ai développée pour réunir plusieurs jeux classiques dans un seul espace interactif. Ce projet avait pour objectif de mettre en pratique mes compétences en développement front-end avec JavaScript et de créer une expérience utilisateur cohérente et ludique.

Le site inclut plusieurs jeux classiques implémentés intégralement en JavaScript :

  • Morpion (Tic-Tac-Toe) : Implémentation classique avec possibilité de jouer contre un autre joueur ou contre l'ordinateur avec plusieurs niveaux de difficulté.
  • Snake : Version moderne du jeu emblématique avec système de score, différentes vitesses et obstacles.
  • Pong : Remake du tout premier jeu vidéo commercial, offrant un mode deux joueurs et un mode contre l'ordinateur.
  • Memory : Jeu de mémoire avec différents thèmes et niveaux de difficulté croissante.
  • 2048 : Adaptation du célèbre jeu de puzzle avec sauvegarde des meilleurs scores.

Fonctionnalités principales de la plateforme :

  • Interface utilisateur unifiée pour accéder à tous les jeux
  • Système de comptes utilisateurs permettant de sauvegarder les scores et préférences
  • Classements globaux pour chaque jeu
  • Design responsive s'adaptant aux différents appareils (ordinateurs, tablettes, smartphones)
  • Personnalisation visuelle (thèmes clairs/sombres, couleurs personnalisables)
  • Mode hors ligne pour jouer sans connexion Internet

Ce projet m'a permis d'approfondir mes connaissances en programmation orientée objet avec JavaScript, en gestion du DOM et en optimisation des performances pour les animations fluides. J'ai également mis l'accent sur l'accessibilité des jeux pour les rendre utilisables avec un clavier, une souris ou un écran tactile.

Technologies utilisées

HTML5 CSS3 JavaScript Canvas API LocalStorage Responsive Design Git

Pour développer GameHub, j'ai utilisé un ensemble de technologies modernes du web :

  • HTML5 et CSS3 : Structure et mise en page moderne, avec utilisation des grid et flexbox pour une interface responsive et adaptative.
  • JavaScript (ES6+) : Développement de tous les jeux en JavaScript pur (vanilla) en appliquant des principes de programmation orientée objet, utilisation de classes, modules et promesses.
  • Canvas API : Utilisation intensive de l'API Canvas HTML5 pour les animations et les rendus graphiques des jeux, particulièrement pour Snake et Pong qui nécessitent un rafraîchissement fluide.
  • Web Audio API : Implémentation d'effets sonores et musiques de fond pour une expérience immersive.
  • LocalStorage : Sauvegarde des scores et des préférences utilisateur en local, permettant la persistance des données entre les sessions.
  • Progressive Web App (PWA) : Configuration pour une utilisation hors ligne et possibilité d'installation sur l'écran d'accueil des appareils mobiles.
  • Webpack : Bundling et optimisation du code pour les performances en production.

J'ai suivi une architecture modulaire pour chaque jeu, permettant une maintenance facile et une évolution du code. L'interface utilisateur a été conçue selon les principes du Material Design, offrant une expérience cohérente et intuitive.

Défis techniques et solutions

Performance des animations

Pour garantir une fluidité des jeux comme Snake et Pong qui nécessitent des mises à jour rapides de l'affichage, j'ai optimisé le code en :

  • Utilisant requestAnimationFrame pour synchroniser le rendu avec le taux de rafraîchissement de l'écran
  • Limitant les manipulations DOM hors de la boucle de jeu
  • Implémentant le double buffering dans Canvas pour réduire le scintillement

Adaptation aux différents appareils

Pour offrir une expérience de jeu optimale sur tous les appareils (PC, mobile, tablette), j'ai mis en place :

  • Des contrôles tactiles pour les appareils mobiles
  • Une détection automatique du type d'appareil pour adapter l'interface
  • Un redimensionnement dynamique des éléments de jeu en fonction de la taille de l'écran

Intelligence artificielle pour les modes solo

Pour créer des adversaires IA engageants dans les jeux comme Morpion et Pong :

  • Implémentation de l'algorithme Minimax pour le Morpion avec différents niveaux de difficulté
  • Développement d'une IA prédictive pour Pong qui analyse la trajectoire de la balle
  • Équilibrage de la difficulté pour offrir un défi progressif aux joueurs
